CSS选择器

  • *{ ... } : 全局选择器 选择全部元素
  • h1{ ... } : 标签选择器 选择所有 h1 元素
  • h1,h2{ ... } : 混合选择器 选择所有满足至少一种条件的元素
  • p span{ ... } : 后代选择器 选择 p 标签下的所有 span 元素
  • ul>li{ ... } : 子代选择器 选择 ul 下的直接子元素 li
  • .class{ ... } : 类选择器 根据class属性选择元素
  • #id{ ... } : id选择器 根据id属性选择元素
  • h1+p{ ... } : 相邻兄弟选择器 选择 h1 元素后的第一个 p 元素
  • h1~p{ ... } : 通用兄弟选择器 选择 h1 元素后的所有同级的 p 元素

CSS弹性盒

  • display : flex; - 父元素
  • flex-direction : 容器内块元素排列方向 - 父元素
    • row (横向排列)
    • column (纵向排列)
    • row-reverse (反向横向排列)
    • column-reverse (反向纵向排列)
  • justify-content : 容器内块元素竖直对齐方式 - 父元素
    • flex-start (上)
    • flex-end (下)
    • center (居中)
  • align-items : 容器内块元素水平对齐方式 - 父元素
    • flex-start (左)
    • flex-end (右)
    • center (居中)
  • flex : 容器内块元素分配空间 - 子元素
    • 语法 flex: flex-grow flex-shrink flex-basis|auto|initial|inherit;
      • flex-grow 一个数字，规定项目将相对于其他灵活的项目进行扩展的量。
      • flex-shrink 一个数字，规定项目将相对于其他灵活的项目进行收缩的量。
      • flex-basis 项目的长度。合法值："auto"、"inherit" 或一个后跟 "%"、"px"、"em" 或任何其他长度单位的数字
      • auto 与 1 1 auto 相同
      • none 与 0 0 auto 相同
      • initial 设置该属性为它的默认值，即为 0 1 auto
      • inherit 从父元素继承该属性
  • align-self : 容器内块元素对齐方式，覆盖父元素的对齐方式 - 子元素
  • flex-wrap : 规定换行 - 父元素
    • nowrap
    • wrap
    • wrap-reverse
  • order : 规定元素排列顺序 - 子元素

CSS浮动

  • float : 浮动方向
    • left
    • right
  • clear : 清除浮动
    • both
    • right
    • left
  • overflow : 可以控制内容溢出元素框时的表现。
    • visible 默认值。内容不会被修剪，会呈现在元素框之外。
    • hidden 内容会被修剪，并且其余内容是不可见的。
    • scroll 内容会被修剪，但是浏览器会显示滚动条以便查看其余的内容。
    • auto 如果内容被修剪，则浏览器会显示滚动条以便查看其余的内容。
    • inherit 规定应该从父元素继承 overflow 属性的值。
    • overflow:hidden;clear:both; 用于浮动元素的父元素时可以清除浮动

CSS定位

  • position : 定位类型
    • relative: 相对定位，参照物是设置了 position 的父元素
    • absolute: 绝对定位，脱离文档流，参照物是设置了 position 的父元素
    • fixed: 固定定位，脱离文档流，参照物是窗口
  • z-index : 调整元素覆盖顺序

    • inherit 规定应该从父元素继承 overflow 属性的值。

  • animation : 执行动画
    • 语法 animation: name duration timing-function delay count dierction;
      • name 关键帧名称
      • duration 持续时间 s ms
      • timing-function 速度设置
        • ease 逐渐变慢
        • ease-in 加速
        • ease-out 减速
        • ease-in-out 先加速后减速
        • linear 匀速
      • delay 延时执行 s ms
      • count 播放次数，infinite表示无限
      • dierction 播放方向
        • normal 向前播放
        • alternate 循环播放
  • animation-play-state : 控制动画播放
    • running 播放
    • paused 停止播放

Date对象

  • getMilliseconds() : 返回毫秒(0-999)
  • getMinutes() : 返回分钟(0-59)
